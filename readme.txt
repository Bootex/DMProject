'2015행복주택 공모전'
행복주택 어디에 설립할 것인가를 데이터 마이닝을 활용하여 풀자라는 생각으로 시작한 프로젝트입니다.
Data 폴더에 버스 데이터와 네이버 검색결과를 파싱하고 전처리하는 파이썬 코드가 들어있습니다.

분석에는 R을 사용하였습니다.
회귀나무 모델을 RPart를 이용하여 구현하였고, 이에 결정된 값을 통해 최종 후보군을 선출하였습니다.
